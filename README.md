Run options
```
Default: 
./rainy_days

Configured:
./rainy_days -config <path> --ascii
```

Example JSON Structure for configuration file
```
{
    "animationInterval": 0.05,
    "rainColors": ["38;5;206m", "38;5;27m"],
    "artColor": "38;5;63m"
}

```
